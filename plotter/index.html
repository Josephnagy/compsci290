Hangman Game

Game Description

This is a hangman game, but with a slight twist. It’s designed to be similar to classic-style hangman, except there is a back-end algorithm making the game more difficult! The objective of the game is to guess the hangman word before running out of guesses. If you run out of guesses, you lose!

How To Play

The first step is to choose a difficulty between easy or hard. The differences between them is easy gives you 16 guesses while hard only gives you 12. After pressing “start game,” you will be prompted to guess a letter in the hangman word. Note: You should only enter lowercase letters as the game will not let you enter anything else! You’ll continue to guess letters until you either win or lose. At the end you will have the option to play again.
